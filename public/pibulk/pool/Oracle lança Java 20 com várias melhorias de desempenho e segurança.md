# Oracle lança Java 20 com várias melhorias de desempenho e segurança

A Oracle lançou o Java 20, uma nova versão da linguagem de programação e plataforma de desenvolvimento Java. O Java 20 traz milhares de melhorias de desempenho, estabilidade e segurança, incluindo aprimoramentos de plataforma que visam melhorar a produtividade dos desenvolvedores e impulsionar a inovação e o crescimento nas organizações. A Oracle apresentou os recursos do Java 20 durante o evento Java Developer Day, parte do Oracle DevLive Level Up, realizado em março de 2023.

Entre as atualizações e melhorias do Java 20, destacam-se:

- Padrões de registro (segunda visualização): permite que os usuários aninhem padrões de registro e padrões de tipo para facilitar a navegação e o processamento de dados de forma poderosa e declarativa.

- Correspondência de padrão para switch (quarta visualização): amplia a correspondência de padrão em instruções switch, permitindo testar uma expressão em vários padrões e executar ações específicas, facilitando consultas complexas orientadas a dados.

- Valores com Escopo (Incubadora): permite o compartilhamento de dados imutáveis entre threads, melhorando a facilidade de uso, compreensão, robustez e desempenho.

- Virtual Threads (Second Preview): simplifica a criação, manutenção e observação de aplicativos simultâneos de alto rendimento, introduzindo threads virtuais leves.

- Simultaneidade Estruturada (Segunda Incubadora): simplifica a programação multithread ao tratar várias tarefas executadas em diferentes threads como uma única unidade de trabalho.

- Função externa e API de memória (segunda visualização): permite que programas Java interoperem com código e dados fora da JVM, chamando funções estrangeiras e acessando com segurança a memória estrangeira.

- Vector API (Fifth Incubator): expressa computações vetoriais que compilam para instruções vetoriais em CPUs suportadas, aumentando o desempenho em comparação com cálculos escalares.

Essas são apenas algumas das atualizações e melhorias do Java 20. A Oracle continua a fornecer novos recursos do Java a cada seis meses por meio de um cronograma de lançamento previsível, oferecendo um fluxo constante de inovações para melhorar o desempenho, estabilidade e segurança da plataforma.
